This project was part of a design sprint between design and program teams in April 2026.

**Overall focus**: Explore how we can intergrate AI into the Simple dashboard.

**Goal**: Program managers can quickly identify quality gap signals from the divisional to the facility level, along with a brief rationale for why those gaps are flagged.
